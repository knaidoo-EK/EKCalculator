# EK Calculator - Script Generating Platform

A Vue.js application for generating medical cannabis prescription scripts with automatic dose distribution based on patient limits.

## Features

- Set daily THC limits (inhaled and oral)
- Add multiple products of each type:
  - THC Flower (dried herb)
  - Non-THC Flower (CBD/CBG)
  - Oral Oil
  - Pastilles/Gummies
  - Vape Cartridges
- Automatic script generation with dose distribution
- Beautiful, modern UI designed for efficiency

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Usage

1. Set your daily THC limits (inhaled and oral)
2. Add products by clicking the "+ Add" buttons
3. Enter product details (name, THC%, or ratio)
4. Click "Generate Scripts" to create prescriptions
5. Copy individual scripts to clipboard
